# 三消算法设计

## 规则
1. 棋盘大小 8*8个格子
2. 初始状态下 有5种棋子
3. 相同类型的棋子连在一起时会消除，横着竖着都可以消，斜着不可以消
4. 棋盘至少要有移动一步能进行消除的事件

## 算法实现

> 1. 动态规划算法
> 2. 三消算法
> 3. 推荐算法
